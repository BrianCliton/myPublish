- **status**: in_progress

## 描述

构建一套服务端-客户端发布系统：
- 服务端使用 ed25519 私钥对发布信息签名
- 客户端内置 ed25519 公钥，验证签名后执行相应行为
- 支持的行为类型：版本更新、地址配置、广告营销等
