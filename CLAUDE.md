## Project Development

This project uses the **PMA skill** (`/pma`) for project management.

### Workflow

All non-trivial changes follow the three-phase workflow:

1. **Investigation** — trace call chains, read related code/config/tests, check changelog
2. **Proposal** — output current state, proposal, risks, scope, alternatives; wait for approval
3. **Implement** — only after explicit approval; verify and record

### Task & Plan Tracking

- Tasks: `docs/task/index.md` → `docs/task/PREFIX-NNN.md`
- Plans: `docs/plan/index.md` → `docs/plan/PLAN-NNN.md`
- Changelog: `docs/changelog.md`
- Architecture: `docs/architecture.md`
